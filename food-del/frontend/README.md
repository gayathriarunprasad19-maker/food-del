## Description
Tomato is a full-stack food delivery web application that allows users to browse food items,
add them to the cart, and place orders. The application demonstrates real-world frontend and
backend integration with a clean and responsive user interface.

## Tech Stack
- Frontend: React.js (Vite), HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB / MySQL
- Tools: VS Code, Git, Postman

## Features
- Food listing by category
- Add to cart and remove from cart
- User authentication popup
- Responsive UI design
- Admin panel for food and menu management

## How to Run the Project
1. Navigate to frontend folder  
2. Navigate to backend folder  


## Project Purpose
This project was developed to gain hands-on experience in full-stack web development and
to understand real-time application flow.
